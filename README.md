# SS-PLRQR
The code of Beyond Spatial Priors: Spectral-Aware Hyperspectral Anomaly Detection Network via Heterogeneous–Homogeneous Super-Band and Quaternionic Sparse Representation

Step1: Extract the files from the compressed folder ss-plrqr to the current directory.

Step2: run A_plan.m (You can change the dataset and training conditions in IN)

X. Wang, Y. Sun, T. Tang and L. Lei, "Beyond Spatial Priors: Spectral-Aware Hyperspectral Anomaly Detection Network via Heterogeneous–Homogeneous Super-Band and Quaternionic Sparse Representation," in IEEE Transactions on Geoscience and Remote Sensing, vol. 63, pp. 1-17, 2025, Art no. 5536217, doi: 10.1109/TGRS.2025.3639603.
