# SS-PLRQR
The code of SS-PLRQR: Super-Spectrum Parallel Low-Rank Quaternion Recovery for Hyperspectral Image Classification

Step1: Extract the files from the compressed folder ss-plrqr to the current directory.

Step2: run A_plan.m (You can change the dataset and training conditions in IN)

L. Qian, X. Wang, M. Hong, Y. Zhang and H. Wang, "SS-PLRQR: Super-Spectrum Parallel Low-Rank Quaternion Recovery for Hyperspectral Image Classification," in IEEE Transactions on Geoscience and Remote Sensing, vol. 63, pp. 1-16, 2025, Art no. 5505216, doi: 10.1109/TGRS.2025.3540460.
